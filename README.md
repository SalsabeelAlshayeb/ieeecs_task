# Git & GitHub Workshop Task

In this task, you will fork this repository and add some changes to it.


## 🎯 Objective
Learn how to:
- Fork a repository
- Clone it locally
- Create a new branch
- Make changes
- Commit and push
- Create a Pull Request

---

1. Fork this repository to your GitHub account
